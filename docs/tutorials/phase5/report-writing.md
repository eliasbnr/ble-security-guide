# Report Writing

Erstellung eines professionellen Security Assessment Reports.

## Struktur

1. **Executive Summary**
2. **Scope & Methodology**
3. **Findings** (mit CVSS)
4. **Attack Scenarios**
5. **Remediation**
6. **Appendix** (PoC, Raw Data)
