# BLE Security Modes

## Security Mode 1 (Encryption)

| Level | Beschreibung |
|-------|--------------|
| 1 | No security |
| 2 | Unauthenticated encryption |
| 3 | Authenticated encryption |
| 4 | LE Secure Connections |

## Security Mode 2 (Signing)

| Level | Beschreibung |
|-------|--------------|
| 1 | Unauthenticated signing |
| 2 | Authenticated signing |
