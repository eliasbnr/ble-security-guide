# Glossar

| Begriff | Erklärung |
|---------|-----------|
| **ATT** | Attribute Protocol |
| **CCCD** | Client Characteristic Configuration Descriptor |
| **Central** | Gerät das Verbindungen initiiert |
| **Characteristic** | Datencontainer in GATT |
| **GATT** | Generic Attribute Profile |
| **Handle** | 16-bit Adresse eines Attributs |
| **LTK** | Long Term Key |
| **MTU** | Maximum Transmission Unit |
| **Peripheral** | Gerät das advertised |
| **Service** | Gruppe von Characteristics |
| **UUID** | Universally Unique Identifier |
