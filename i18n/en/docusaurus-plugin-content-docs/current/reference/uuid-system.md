# UUID System

UUIDs identify services, characteristics, and descriptors.

## 16-bit vs 128-bit

**16-bit** (Bluetooth SIG reserved):
```
0x1800 → Generic Access
0x180F → Battery Service
```

**128-bit** (Custom):
```
d44bc439-abfd-45a2-b575-925416129600
```
